# 14 - Frontend Minimal UI

## Goal
A neat, minimal, interactive first UI.

## Current features
- agent dropdown
- printer dropdown
- PDF file picker
- print button
- recent jobs list with live refresh

## Run
```bash
cd /home/vimal/.openclaw/workspace/printbridge
npm install
npm run dev:frontend
```

## Requires
- backend running on port 4000
- Windows agent running and synced
