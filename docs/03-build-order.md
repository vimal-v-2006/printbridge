# 03 - Build Order

## Phase 1 - Foundation
1. Create monorepo structure
2. Define shared data models
3. Define API endpoints
4. Decide frontend/backend/agent stacks

## Phase 2 - Backend MVP
1. Agent registration endpoint
2. Printer sync endpoint
3. File upload endpoint
4. Create print job endpoint
5. Job status endpoint

## Phase 3 - Agent MVP
1. Agent bootstrap config
2. Heartbeat to backend
3. Printer discovery
4. Accept print jobs
5. Execute print through OS
6. Return status/errors

## Phase 4 - Frontend MVP
1. Login screen or simple access gate
2. Dashboard with agents/printers
3. Drag-and-drop upload
4. Print options panel
5. Queue/history screen

## Phase 5 - Polish
1. Better status feedback
2. Better printer capability detection
3. Retry/cancel controls
4. Cleaner design system
5. Installer/package flow for agent
