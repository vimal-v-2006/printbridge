# Frontend

Minimal interactive web UI for PrintBridge.

## What it does
- fetches agents and printers from backend
- uploads a PDF
- creates a print job
- shows recent job status

## Run

```bash
npm run dev:frontend
```

Open:
- `http://localhost:5173`

## Notes
- expects backend at `http://localhost:4000`
- currently optimized for a simple local prototype
